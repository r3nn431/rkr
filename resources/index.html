About the development

This game came from a long development, longer than necessary. It all started with a game I made in Twine (interactive text) because I really wanted to make a simple RPG game but it still wasn't enough. That's when I started doing it in Java, but I had some problems because I was still learning. After a few more attempts with other languages ​​and platforms, the game finally came out using NeutralinoJS.

I was heavily inspired by Kingsway when making this game, which I really advise everyone to play as it's a wonderful game. This game is not perfect and it's not a game that will get you addicted, but it's made with love, born from a hobby that I really enjoyed being able to do and finish.
